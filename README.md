## 🛠️ Getting Started

### Prerequisites
- Node.js 18+
- npm 9+

### Installation

```bash
# Clone the repository
git clone https://github.com/YOUR_USERNAME/emoji-digitals.git

# Navigate into the project
cd emoji-digitals

# Install dependencies
npm install

# Start development server
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

### Build for Production

```bash
npm run build
```

Output goes to the `dist/` folder as a single `index.html` file.

### Preview Production Build

```bash
npm run preview
```

## ✨ Features

- 🌑 **Dark / Light mode** toggle with localStorage persistence
- 🎬 **Live code animation** in hero section (Canvas API)
- 🌐 **Neural network footer** animation (Canvas API)
- 📱 **Fully responsive** — mobile, tablet, desktop
- ⚡ **Single file build** — entire app bundles into one `index.html`
- 🔄 **Client-side routing** with React Router
- 🖼️ **Smart image loading** — Google Drive images with graceful fallback
- 🎓 **Academy page** with course listings, certifications banner, training gallery
- 🔬 **Innovation Lab** page with experiment showcases
- 📊 **Case Studies** with animated metrics
- 🛠️ **Tech Stack** explorer with skill bars
- ⚙️ **Interactive Process** timeline
- ❓ **FAQ** accordion with category tabs
